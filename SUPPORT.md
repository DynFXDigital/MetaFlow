# Support

## Getting help

- **Usage questions / troubleshooting:** GitHub Discussions
- **Bug reports:** GitHub Issues using the Bug Report template
- **Feature requests:** GitHub Issues using the Feature Request template
- **Security issues:** Follow private reporting instructions in `.github/SECURITY.md`

## Before opening an issue

- Confirm you are on a recent version.
- Include minimal reproduction steps.
- Include OS, VS Code version, extension version, and relevant logs.

## Maintainer response expectations

- Best-effort triage for new issues weekly.
- Security reports follow the response targets in `.github/SECURITY.md`.
