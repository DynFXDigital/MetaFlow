# Plan: Bundled Metadata/Tools/MCP (Retired)

This plan is retired as of 2026-03-01.

MetaFlow intentionally de-scoped runtime bundled metadata composition and MCP onboarding from the core extension. The current approach is:

1. Keep MetaFlow focused on deterministic overlay/materialization from user-configured metadata sources.
2. Provide a one-time template workflow via `MetaFlow: Initialize MetaFlow AI Metadata`.
3. Avoid extension-managed runtime settings/contracts for bundled baseline and MCP onboarding.

Historical phase notes were removed to avoid stale implementation guidance.
