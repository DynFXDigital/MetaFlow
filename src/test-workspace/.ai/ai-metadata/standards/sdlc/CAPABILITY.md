## Mission

Provide reusable SDLC traceability metadata that can be layered into repositories using MetaFlow.

## In Scope

- Traceability-focused instructions.
- SDLC-oriented skills.
- Agent guidance tied to testing and validation.

## Non-Goals

- Repository-specific build or deployment policy.
- Runtime code changes outside metadata composition.
